# 🚀 ProjectFlow - Project Management Tool

A modern full-stack Project Management Tool that helps users organize projects, manage tasks, collaborate with teams, and track progress through an intuitive dashboard.

## 🌐 Live Demo

**Frontend:** https://projectflow-client.vercel.app

---

## ✨ Features

- 🔐 Secure User Authentication (JWT)
- 📁 Create, Update & Delete Projects
- ✅ Task Management
- 👥 Team Management
- 📅 Calendar View
- 📊 Reports Dashboard
- 🛡️ Protected Routes
- 📱 Responsive Design
- 🎨 Modern UI

---

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ansariosama024-oss/codsoft_tasks.git
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the `server` folder.

```env
PORT=5000
MONGODB_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET_KEY
JWT_EXPIRES_IN=7d
CLIENT_URL=http://localhost:5173
```
